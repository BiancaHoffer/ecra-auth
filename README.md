Ecrã de login, com um formulário, que simule um pedido ao backend bem sucedido e resulte na entrada ou não entrada e um pedido mal sucedido (exemplo: password errada, etc.

Principais tecnologias utilizadas
- ReactJS;
- Vite;
- TypeScript;
- React Hook Form;
- Zod;
- Styled-components.
